# ব্যাসিক ইনপুট আউটপুট

পাইথন এ ডাটা ইনপুট এবং আউটপুট এর জন্য আমরা স্পেশাল কিছু ফাংশন ব্যবহার করে থাকি।
ইনপুট এবং আউটপুট এর বেসিক ফাংশন গুলো হল :

- `input()` (python3)
- `raw_input()` (python2)
- `print()`

## ইনপুট ফাংশন `input()` এবং `raw_input()`

এই দুটি ফাংশন আমরা সাধারণত ইউজার এর কাছ থেকে ইনপুট নেয়ার জন্য
ব্যবহার করে থাকি। `input()` এবং `raw_input()` ফাংশনের ব্যবহারের
নিয়ম অনেকটা একই ধরণের। উদাহরণ দিলে ব্যাপার টা আরো পরিষ্কার হবে আশা করি।
```python
a = input()
```
এ ক্ষেত্রে পাইথন ইন্টারপ্রেটার যখন কোড টা এক্সিকিউট করবে তখন
ইন্টারপ্রেটার ইউজার এর ইনপুট এর জন্য অপেক্ষা করবে এবং ততক্ষণ
পর্যন্ত ইনপুট এর ডাটা গ্রহণ করবে যতক্ষণ না ইউজার `Enter` বাটন
প্রেস করে অথবা ইনপুটে আরেকটি নিউলাইন ক্যারেক্টার আসে।
আমাদের মাথায় রাখতে হবে এক্ষেত্রে আমরা ইউজার থেকে যেই ডাটা নিলাম
তা আমাদের `a` ভ্যারিয়েবলে একটি স্ট্রিং হিসেবে থাকবে। আমরা এই
ইনপুট ডাটা কে অন্য ডাটা টাইপে পরিবর্তন করার জন্য পাইথনের বিভিন্ন
ফাংশন ব্যবহার করতে পারি। যেমনঃ `int`, `float` ইত্যাদি।

বিঃদ্রঃ পাইথন ২ এর জন্য আমরা `input` এর পরিবর্তে `raw_input`
ব্যবহার করব।

## আউটপুট ফাংশন `print()`

`print()` ফাংশন কেবলমাত্র তাই আউটপুট দেয় যা এর আর্গুমেন্ট হিসেবে
দেয়া হয়। কয়েকটি উদাহরণ দেখলে আউটপুট এর ব্যাপারটি আরো সহজ হবে।
```python
print('Hello World!')
```
উপরের প্রোগ্রামের আউটপুট হবে যথারীতি `Hello World!`।

আমরা আমাদের প্রোগ্রামের বিভিন্ন ভ্যারিয়বল ও এই ফাংশনের মাধ্যমে
প্রিন্ট করতে পারি।
```python
a = 'Hello World!'
b = 45

print(a)
print(b)
```
উপরের কোডের আউটপুটঃ
```sh
Hello World!
45
```
আরো একভাবে আমরা প্রিন্ট ফাংশনে আমাদের ডাটা আউটপুট দিতে পারিঃ
```python
name = input()

print('Hello {}'.format(name))
```
উপরের কোডের ক্ষেত্রে আমরা ইউজার প্রথমে তার নাম ইনপুট করবেন।
এরপর ঐ নাম আমরা স্ট্রিং এর `format` মেথড ব্যবহার করে সহজেই
প্রিন্ট করতে পারব। এখানে যদি আমরা ইনপুট দেইঃ `Pythonista` তাহলে
আমাদের কোডের আউটপুট হবেঃ
```sh
Hello Pythonista
```

আশা করি পাইথনের ইনপুট এবং আউটপুট এর ফাংশন গুলো বুঝতে
পেরেছেন। আর এই ফাংশন গুলো সম্পর্কে আরো বিস্তারিত জানার জন্য
আপনি [পাইথনের অফিশিয়াল ডকুমেন্টেশন](https://docs.python.org/3/library/functions.html) ভিজিট করতে পারেন।
